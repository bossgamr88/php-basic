เขียนโปรเเกรมเเบบโครงสร้าง Structure Programing 
โปรเเกรมย่อย ช่วยให้ จัดการต่างๆ ง่ายขึ้น ไม่เป็นการ เพิ่ม ลบ เเก้ไข โปรเเกรม โปรเเกรมย่อย เเบ่ง หน้าที่การทำงาน อย่างชัดเจน
เเละ ทำงานลักษณะซ้ำ ได้ทันทีเพียงเรียกเรียกใช้

รูปเเบบ ประกาศ function ใน PHP

function ชื่อฟังก์ชั่น([argument1,argument2,...])
{
	Statement;	
}

argument เป็นตัวเเปร ที่ตั้งขึ้นมา เพื่อรอรับค่า ที่ฟังก์ชั่นส่งผ่านค่าเข้ามาประมวลผลภายใน ฟังกชั่น
ตั้งกี่ตัวก็ได้ ถ้าไม่มี รับค่า ไม่ต้องระบุ ก็ได้ 

Statement เป็นคำสั่งประมวลผลภายใน ฟังก์ชั่น
